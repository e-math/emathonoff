EmathEquationarray

Create equationarrays in web

This jQuery-plugin is used to show a mathematical LaTeX-like equationarrays in both viewmode and editmode. It depends on jQuery and MathQuill. Optionally mqpanel.js can also be used.

Here are couple of examples, how to use jquery.emathequationarray.js.

Non-editable

Using emathequationarray to show existing equation array

If needed, for example for exercises, some fields can be defined as editable.

<div id="eqnarr1"></div> <script type="text/javascript"> jQuery('#eqnarr1').emathequationarray({ "editable": false, "eqnarray": [ { "left": "3x+1", "middle": "=", "right": "4x-5", "editmodes": { "left": true, "middle": false, "right": false } }, { "left": "3x", "middle": "=", "right": "4x-9", "editmodes": { "left": false, "middle": true, "right": false } }, { "left": "-x", "middle": "=", "right": "-9", "editmodes": { "left": false, "middle": false, "right": false } }, { "left": "x", "middle": "=", "right": "9", "editmodes": { "left": false, "middle": false, "right": false } } ] } </script>

Editable

Using emathequationarray to edit an equation array

In editmode there are couple of keyboard shortcuts:

ctrl+arrows
Move in fields
enter
Go to next field. (In the last fields, add a new row and go to next field.)
ctrl+enter
Add a new empty row after current row.
shift+enter
Add a copy of current row after current row.
ctrl+backspace
Remove current row.
<div id="eqnarr2"></div> <script type="text/javascript"> jQuery('#eqnarr2').emathequationarray({ "editable":true, "authormode": false, eqnarray: [ {left: "3x+1", middle: "=", right: "4x-8"}, {left: "3x", middle: "=", right: "4x-9"}, {left: "-x", middle: "=", right: "-9"}, {left: "x", middle: "=", right: "9"} ] }); </script>
